    Weaving and knitting - industrial progression and computation

    What is a punchcard?

    What started with the automatization of loom weaving with the Jacquard loom and its' operative punchcards to inform of treadle changes, the punchcard system has moved towards popular use in machine knitting, while project drafting for weaving bears resemblance to the binary system used in early computation. Alongside manipulating the knitting machine manually to create textures and colour changes, you can help the machine know what to do and when to do it by itself with the use of a punchcard. Depending on the technique you would like to use, whether you punch a hole or leave it unpunched informs the machine of different things, always interacting with other machine functions that you set up yourself.
